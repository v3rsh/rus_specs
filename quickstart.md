# Руководство по быстрому старту

## Порядок выполнения команд

1. **Инициализация проекта** (в терминале):
   ```bash
   specify init <PROJECT_NAME>
   # или
   specify init .
   ```

2. **Определение конституции** (в чате AI-агента):
   ```markdown
   /speckit.constitution [ваши принципы проекта]
   ```

3. **Создание спецификации** (в чате):
   ```markdown
   /speckit.specify [описание того, что вы хотите построить]
   ```

4. **Уточнение спецификации** (в чате, опционально):
   ```markdown
   /speckit.clarify [области для уточнения]
   ```

5. **Проверка спецификации** (в чате, опционально):
   ```markdown
   /speckit.checklist
   ```

6. **Создание технического плана** (в чате):
   ```markdown
   /speckit.plan [ваш технологический стек и архитектурные решения]
   ```

7. **Создание списка задач** (в чате):
   ```markdown
   /speckit.tasks
   ```

8. **Анализ плана** (в чате, опционально):
   ```markdown
   /speckit.analyze
   ```

9. **Реализация** (в чате):
   ```markdown
   /speckit.implement
   ```
